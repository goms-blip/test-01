# AI 학습 조교 대화 기록

## Q: 지난 수업에서 배운 API 라우팅 방식이 뭐였지?

지난 수업에서는 REST API의 개념과 HTTP 메서드를 이해하고, Express.js를 활용한 기본 라우팅 구현에 대해 배웠습니다. REST API는 리소스 중심으로 설계되며, HTTP 메서드(GET, POST, PUT, DELETE 등)를 사용하여 행위를 구분합니다. Express.js에서는 `app.get()`, `app.post()` 등의 메서드를 사용하여 라우트를 정의할 수 있습니다. 또한, 미들웨어를 사용하여 공통 기능(파싱, 인증, 로깅 등)을 처리할 수 있습니다.
