## 사용 모델

- **OpenAI gpt-image-1** (사용자가 "gpt image2"로 지칭. 정식 모델명은 `gpt-image-1`)
- 메인/빌보드: `/v1/images/generations` (text-to-image)
- 캐릭터 포스터: `/v1/images/edits` (image+text-to-image, 스틸 `cineplay_06_hyeonguk_lee-hyoje.jpg` 입력)
- quality: `high`, n: 1
- API 키는 환경변수로만 사용. 파일·메모리에 평문 저장 안 함.

## 디자인 의도 (INFO.md §6, §9 반영)

1. **빨간 카운트다운/폰 글로우** = 핵심 비주얼 훅 → 두 컷 모두 적색 글로우를 단일 발광원으로 강조
2. **무표정 클로즈업 vs 공포 광원 대비** → 차가운 청록 형광등 + 위험한 적색의 보색 대비
3. **5인 군상극 + 외부자(빨간 폰의 그림자)** 구도 → 빌보드 가로형 우측 뒤에 그림자 6번째 인물 배치
4. **앱 UI / LCD 타이머 모티프** → 타이틀 폰트를 글리치 LCD 디스플레이 톤으로 지시
5. **무속 부적 모티프** → 폰 화면·벽 위에 미세하게 부적 패턴 암시

## 알려진 이슈 / 다음 단계

- **한글 타이틀 렌더링 불완전**: 두 장 모두 "기리고"의 마지막 "고" 자가 "ㄱ"·"ㅁ" 등으로 어긋남. 이미지 모델의 한글 합성 한계.
  - 권장 후처리: Figma/Photoshop에서 **실제 한글 폰트**(예: Noto Sans KR Black, 산돌 격동고딕, 본고딕)로 타이틀 텍스트만 마스킹·재합성.
  - 또는 본문 영역만 AI 생성 → 타이포그래피는 별도 레이어로 합성하는 워크플로우 추천.
- **Netflix 트레이드마크**: 세로 컷 하단에 "NETFLIX" 워드마크가 자동 생성되었음. 실제 게재 시 라이선스 확인 필요. 개인/포트폴리오 컨셉용으로만 사용 권장.
- **얼굴/캐릭터 일치**: 실제 배우(전소영·강미나·백선호 등) 얼굴은 의도적으로 비슷하게 만들지 않음(초상권). 익명 컨셉 컷.

## 보안 메모

- 본 세션에서 사용자가 OpenAI API 키를 평문으로 채팅에 공유함.
- **즉시 키 폐기 및 재발급 권장.** (platform.openai.com → API keys → Revoke)
- 키는 어떤 파일에도 저장되지 않음. 셸 히스토리(`~/.zsh_history`)에는 잔존 가능 → `history -c` 또는 해당 라인 수동 삭제 권장.
