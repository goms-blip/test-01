

# 🥕 프로젝트: 당근마켓 클론 (Carrot Market Clone)

## 1. 프로젝트 개요본 프로젝트는 중고 거래 플랫폼인 '당근마켓'의 핵심 기능을 클론 코딩하며, 
**인증(Auth), 데이터베이스(DB), 스토리지(Storage), 실시간 채팅** 기능을 하나의 서비스로 통합하는 것을 목표로 합니다.

## 2. 핵심 목표
- **기능 통합:** 부트캠프에서 학습한 Auth, DB, 이미지 업로드, 채팅 기능을 한 앱에 구현.
- **실제 서비스 경험:** 단순 구현을 넘어 Vercel 배포 및 타인과의 실시간 인터랙션 검증.
- **RLS 활용:** Supabase의 Row Level Security를 통해 데이터 보안 권한 설정 습득.

## 3. 주요 기능 및 요구사항 (User Stories)
### Part 1: 인증 및 사용자 설정
- **회원가입/로그인:** 이메일을 통한 가입 및 로그인.
- **동네 설정:** 사용자의 활동 지역 설정 (직접 입력 또는 위치 인증).

### Part 2: 중고 거래 게시판 (CRUD)
- **상품 등록:** 
  - 이미지(최대 3장) 업로드 기능.
  - 제목, 가격, 설명, 카테고리 정보 포함.
  - **상품 관리:** 본인이 작성한 글에 대해서만 수정 및 삭제 가능 (RLS 적용).
  - **조회 및 검색:**
  - 최신순 목록 리스트.
  - 카테고리별 필터링 및 키워드 검색 기능.
  - **상세 페이지:** 이미지 슬라이드, 판매자 정보 및 '관심' 버튼.

### Part 3: 실시간 1:1 채팅- **채팅 시작:** 상품 상세 페이지에서 '채팅하기' 버튼으로 판매자와 연결.
- **실시간성:** Polling 또는 DB 변화 체크를 통한 실시간 메시지 송수신.

### Part 4: 마이페이지- **활동 내역:** 내가 등록한 상품, 관심 등록 상품, 참여 중인 채팅 목록 확인.

## 4. 기술 스택
- **Framework:** Next.js (권장)
- **Deployment:** Vercel
- **Backend/Infrastructure:** Supabase
  - Auth: 사용자 인증
  - Database: 상품, 사용자, 채팅 데이터 저장
  - Storage: 상품 이미지 저장
  - Realtime: 실시간 채팅 기능 구현

## 5. 프로젝트 구조 (Workflow)

[가입/로그인] → [동네 설정] → [상품 등록/목록 검색] → [상세 정보 확인] → [1:1 채팅] → [마이페이지 관리]


## 6. 마일스톤 및 우선순위
1.  **Phase 1 (필수):** 회원가입, 상품 등록 및 목록/상세 페이지 구현.
2.  **Phase 2 (핵심):** Vercel 배포 및 기본 기능 동작 확인.
3.  **Phase 3 (고급):** 1:1 실시간 채팅 기능 및 RLS 보안 설정.
4.  **Phase 4 (검증):** 지인 초대 및 실제 가입/채팅 테스트.
