- **핵심 가치**: 복잡한 기능 없이도 사용자가 [어떤 문제/니즈]를 즉각적으로 해결할 수 있게 함

## 3. 핵심 기능 (Key Features - Part 2)
### 3.1 [핵심 기능 이름] (Core Mission)
- **상태**: 구현 및 동작 완료 (v1의 유일한 목표)
- **상세 내용**: 
  - 재료 검색: 재료 이름(한글/영문)으로 즉시 해당 재료 페이지로 이동
카테고리 탐색: 가루류 / 팽창제 / 당류 / 유제품·지방 / 기타(계란·견과·초콜릿·향료) 5개 카테고리 브라우징
재료 상세 페이지: 재료 1개당 9개 정보 항목을 정형화된 형태로 제공
비교 모드: 헷갈리는 재료 2~3개(예: 박력분/중력분/강력분, 베이킹소다/베이킹파우더)를 한 화면에 옆으로 펼쳐 9개 항목을 나란히 비교. 차별화 포인트의 핵심 기능
  - 레시피를 보다가 모르는 재료 이름이 나와 매번 따로 검색하는 데 시간을 쓰는 사람
박력분/중력분/강력분, 베이킹소다/베이킹파우더처럼 헷갈리는 재료를 정확히 구별하고 싶은 사람
한국에서 직접 재료를 구매하고 베이킹을 시도하려는 초보자
  - **제외 사항**: 부가 기능(설정, 프로필 등)보다 핵심 로직 완성에 집중
