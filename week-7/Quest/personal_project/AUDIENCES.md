## 0. 한 줄 포지셔닝 (메시징 코어)

세 페르소나 모두에게 공통으로 꽂아야 할 메시지.

- **For**: 한국 베이킹 초보자
- **Who**: 레시피의 모르는 재료(박력분/베이킹파우더/타르타르 크림 등) 한 줄에 막혀 30분씩 검색을 헤맨다
- **Our app**: 30개 핵심 재료를 9개 항목 표준 구조로 정리한 한국형 비교 사전
- **Unlike**: 흩어진 블로그·유튜브 답변, 영문 위주 해외 사전, 매번 다시 물어야 하는 LLM
- **We provide**: 검색창 → 답까지 3초 + 헷갈리는 재료를 한 화면에서 옆으로 비교 + 한국 마트 구매처

광고/CTA 카피로 바로 재활용 가능한 짧은 변형:
- 메인 헤드라인: "박력분이 뭔지 3초 안에."
- 인스타 후크: "베이킹 첫 실패는 재료 이름에서 시작된다."
- 카페 답변용 서명: "헷갈리는 재료, 한 화면에 펼쳐서 보여주는 사전 만들었어요 → [링크]"

## 1. 타겟 페르소나 3명

각 페르소나는 가상이지만 실제 존재할 법한 인물. 시간·장소·직전 행동까지 박힌 실패 장면 1개씩 포함.

### 페르소나 A — 민지 (27세, 회사원, 1인 가구)

- **상황**: 강남 IT 중소기업 마케터. 평일 야근 2~3회. 인스타에서 본 마들렌 사진에 끌려 한 달 전 베이킹 도구 세트(오븐 미니·계량컵·실리콘 몰드)를 14만 원에 구매. 인스타에 "베이킹 시작" 게시물 1장 올린 직후 멈춤.
- **지금 겪는 문제 (실패 장면)**: 토요일 오후 2시, 자취 원룸 주방. 인스타 저장한 마들렌 레시피를 켰다. "박력분 100g, 베이킹파우더 3g"에서 멈춤. 네이버에 "박력분이 뭐예요"를 치고 블로그 3개를 30분 동안 읽다 결국 동네 마트로 나가 그냥 "밀가루"라고 적힌 봉지를 사 옴. 식감이 푸석. 인스타에는 결과물을 올리지 못함. 그날 밤 "재능이 없다"고 결론 내고 도구는 싱크대 아래 서랍으로.
- **앱이 풀어주는 것**: 한 검색으로 "박력분 = 단백질 함량 8% 이하의 케이크용 가루" + "중력분과의 차이" + "마트에서 어디 매대" + "없으면 중력분 70% + 옥수수전분 30%로 대체 가능"까지 3초. 실패 회피 → 재시도 → 인스타 인증샷.
- **앱을 켜는 트리거**: 레시피 보다 모르는 재료가 나오는 순간. 빈도 — 시도 시 평균 4~6회.

### 페르소나 B — 수현 (35세, 일하는 엄마 + 6세 딸)

- **상황**: 출판사 5년 차 편집자, 재택+사무 혼합. 6세 딸과의 주말 활동을 핀터레스트·인스타 "엄마와 아이 베이킹" 보드에서 매주 큐레이션함. 베이킹 도구는 어느 정도 갖춤(전기오븐, 핸드믹서). 하지만 실제 성공률은 절반 이하.
- **지금 겪는 문제 (실패 장면)**: 일요일 오전 11시, 거실 식탁 위. 딸이 앞치마를 묶고 옆에 서 있다. 레시피 "타르타르 크림 1/2tsp"에서 막힘. 휴대폰을 들고 검색을 시작하자 딸이 "엄마 언제 만들어?"를 외치고 5분 만에 거실 카펫으로 이탈. 휴대폰을 더 보다 보면 "엄마는 폰만 본다"는 죄책감 + 오븐 예열 시간 초과로 그날 베이킹은 무산. 결국 마트 베이커리 컵케이크로 대체.
- **앱이 풀어주는 것**: 휴대폰을 30초 안에 닫을 수 있는 답. "타르타르 크림 = 머랭 안정제. 없으면 레몬즙 1/4tsp 또는 식초 1/4tsp으로 대체"가 한 화면. 딸이 떠나기 전에 다시 베이킹으로 복귀.
- **앱을 켜는 트리거**: 아이 옆에서 손을 놓을 수 없는 시간 압박 + 모르는 재료. 빈도 — 주말 평균 1~2회.

### 페르소나 C — 지훈 (24세, 대학생 자취, 학교 근처 원룸)

- **상황**: 서울 소재 4년제 대학 3학년, 시험 기간 스트레스 해소로 베이킹 유튜브 시청. 원룸 주방에 미니 오븐(전자레인지 겸용) + 핸드믹서 1개. 재료 예산 월 3만 원 이내. 브라우니·드롭쿠키·바나나브레드처럼 단순 디저트 위주.
- **지금 겪는 문제 (실패 장면)**: 시험 전날 화요일 밤 11시, 원룸 책상. 유튜브 "초간단 브라우니" 영상을 보다 "무염버터 100g"에서 막힘. 냉장고에는 가염버터만 있음. "이거 그냥 써도 되나?" 검색 → 블로그·유튜브 5분 → 결론 못 냄 → 배달앱으로 던컨 도넛 주문. 베이킹 시도 0회로 그 주 종료.
- **앱이 풀어주는 것**: 재료 페이지 "대체 가능 여부 + 비율" 항목 + 비교 모드("무염버터 vs 가염버터")로 0.5번 결심에서 1번 실행으로. 비싼 도구·재료 추가 구매 없이 가진 것으로 시도.
- **앱을 켜는 트리거**: 가진 재료로 대체 가능한지 즉답이 필요할 때. 빈도 — 시도 시 평균 2~3회.

### 세 페르소나가 공통으로 막히는 지점

| 공통 페인 | 앱 매핑 항목 |
|---|---|
| 이름이 낯설어 검색이 길어진다 | 검색창 + 한 줄 정체 + 사진 |
| 비슷한 재료 차이를 모르겠다 | 비교 모드 (박력/중력/강력, 베이킹소다/파우더) |
| 가진 재료로 대체 가능한지 | 항목 7 — 대체재 + 비율 |
| 한국 마트 어디서 사는지 모름 | 항목 9 — 한국 구매처 |

## 2. 그들이 모이는 곳 3곳

세 페르소나가 실제로 시간을 보내는 온라인 공간. 진성 답변·콘텐츠로 접근 가능한 채널 위주. 유료 광고 없음.

### 채널 1 — 네이버 카페 (베이킹 초보 질문 게시판)

- **대표 카페 후보 (가입 직후 회원수·등업 조건 사전 확인 필수)**:
  - "쉬운베이킹 클래스" — 회원수 수만 단위, 초보 질문 게시판 활성
  - "이밥차 베이킹스쿨" — 레시피 + 질문 혼합, 답변 문화 강함
  - "홈베이킹 노하우" / "당기는 베이킹" 등 — 보조 후보
- **왜 거기**: 페르소나 A·B가 "박력분 사도 되나요?", "베이킹파우더 대체 뭘로 하나요?" 류 질문을 직접 올리는 1차 접점. 답변에 링크 한 줄이 따라가도 자연스러움 (스팸 아닌 한). 검색 SEO도 같이 따라옴 (네이버 카페 글이 구글에도 노출).
- **신뢰를 사는 글 형태**: ① 질문에 진짜 답을 먼저 4~5줄로 적고, ② "더 자세히 정리한 사전 만들었어요" 한 줄로 링크 첨부. 답변 본문이 링크보다 길어야 함.
- **반드시 확인할 룰**: 등업 조건(가입 후 출석·댓글·게시글 점수), 외부 링크 허용 여부, 자기 사이트 홍보 금지 조항. 첫 답변 전에 카페 공지 5개를 먼저 읽는다.

### 채널 2 — 인스타그램 해시태그 + Reels

- **타겟 해시태그 (한국어)**: `#홈베이킹` `#베이킹초보` `#베이킹입문` `#홈베이커` `#베이킹스타그램` `#오븐없이베이킹` `#박력분` `#베이킹재료` (재료별 변형 — 재료 카드 1장당 해당 재료 해시태그 추가)
- **왜 거기**: 페르소나 A·B가 "오늘 뭐 만들지" 결심 직전 시간을 보내는 곳. 사진·카드뉴스·짧은 Reels에 강함. MISSION의 "시각 통일감(차별화 #4)"이 그대로 채널 적합도로 이어짐.
- **신뢰를 사는 콘텐츠**: 30개 재료 = 카드 1장씩 = 30일 시리즈. 카드 1장 구성 — 재료 사진(통일 톤) / 한 줄 정체 / 헷갈리는 재료와의 차이 1줄 / "전체는 [링크]". Reels 보조 — "박력분 vs 중력분 5초 비교" 같은 5초 영상.
- **반드시 확인할 룰**: 인스타 프로필 링크는 1개만 가능 → linktr.ee 또는 직접 도메인. 모든 카드는 "더 보기"에 링크 명시 + 첫 댓글에 한 번 더.

### 채널 3 — 유튜브 베이킹 채널 댓글 + 카카오 오픈채팅

- **유튜브 (페르소나 C·A 도달)**:
  - 1순위: 아내 채널 (전략 1과 직접 연결)
  - 2순위: 한국 홈베이킹 유튜버 영상 댓글창. 특히 "박력분 vs 중력분", "베이킹파우더 vs 베이킹소다", "초보자가 자주 틀리는 재료" 류 영상의 최신 댓글창이 노출 핫스팟. 검색어 예: "박력분 차이", "베이킹 초보 재료"
  - 접근 방식: 영상 본문에 보강할 정보 1개를 댓글로 적고, 마지막에 "관련 30개 재료를 한 화면에 비교해뒀어요 → [링크]"
- **카카오 오픈채팅 (페르소나 B·A 도달)**:
  - 검색 키워드: "홈베이킹", "베이킹 초보", "베이킹 모임", "베이킹 정보방"
  - 활성방 골라내는 기준: 인원 30명 이상 + 최근 24시간 메시지 5건 이상
  - 접근 방식: 입장 → 자기소개 1문장 → 며칠은 답변만 → 적절한 질문에 답변하면서 사이트 첨부. 광고 1회로 강퇴되는 방이 많으므로 도배 절대 금지.
- **왜 거기**: 유튜브는 페르소나 C가 베이킹 결심 직전 머무는 곳. 카카오 오픈챗은 페르소나 B(워킹맘)가 정보 교환에 의지하는 비공개 허브.

### 사용하지 않을 채널 (의도적 제외)

| 채널 | 이유 |
|---|---|
| 디시인사이드 / 에브리타임 | 베이킹 초보 밀도 낮고, 진성 답변 톤이 안 맞음 |
| X(Twitter) | 한국 베이킹 초보 클러스터 약함 |
| 페이스북 그룹 | 한국 활성도 낮음 |
| 유료 광고(메타·구글) | v1 수익 모델 없음 → ROAS 음수 |

## 3. 첫 10명 획득 전략 3가지

3주(D-7 ~ D+14) 안에 첫 10명을 데려올 행동 단위. 각 전략은 "행동 / 타임라인 / 예상 도달 / 비용 / 위험·금지선" 5칸.

### 전략 1 — 아내 유튜브 채널 1회 자연 언급 (D+0)

- **행동**: 데모데이 직후 아내가 올리는 베이킹 영상 1편의 도입 또는 마무리 5초 안에 "초보자분들 재료 헷갈릴 때 이 사이트 참고하세요" 한 마디. 영상 설명란 첫 줄 + 고정 댓글에 도메인 + 한 줄 소개("Every baking ingredient, explained simply") + UTM(`?utm_source=youtube&utm_medium=wife&utm_campaign=launch`) 부착.
- **타임라인**: D+0 (영상 업로드 당일). 이전 D-3까지는 아내와 노출 강도·카피 합의 완료.
- **예상 도달**: 아내 채널 영상 평균 조회 × 설명란 클릭률 3~5% 가정. 첫 24시간 안에 신규 방문 10~30명, 그중 첫 10명 달성 가능성 가장 높은 채널.
- **비용**: 0원.
- **위험·금지선**: 아내 채널 톤 훼손 금지. "이 앱 만들었으니 써주세요" 식 자기 PR 어조 금지 — 아내 시청자 신뢰가 핵심 자산. 합의 안 된 노출 강도는 시도하지 않음. 영상 1편 1회로 제한, 도배 금지.

### 전략 2 — 인스타 카드뉴스 "30일 재료 시리즈" (D-7 ~ D+22)

- **행동**: 데모데이 일주일 전부터 매일 1장 인스타 카드뉴스 발행. 카드 1장 = 재료 1개. 구성 — 재료 사진(통일 톤) / 한글 + 영문 이름 / 한 줄 정체 / 헷갈리는 재료와의 차이 1줄 / 캡션 마지막 "전체 30개 재료는 프로필 링크" + 재료 해시태그 5개. 30일 후 자동으로 30장 시리즈 완성. Canva 무료 템플릿 1종 고정해서 디자인 일관성 유지(첫 5장으로 템플릿 확정).
- **타임라인**: D-7 (사전 발행 시작) → D+0 (사이트 오픈) → D+22 (30장 완성).
- **예상 도달**: 카드 1장당 도달 50~150명(해시태그 신규 계정 기준), 클릭률 1~2%로 신규 방문 1~3명. 30일 누적 신규 방문 30~90명, 그중 첫 10명 달성은 D+5~D+10 사이 예상.
- **비용**: 0원 (Canva 무료 + 본인 이미지 자산 = Supabase Storage에 이미 올린 30장 재료 사진 재활용).
- **위험·금지선**: 디자인 톤 흔들리면 시리즈성 훼손 → 첫 5장 후 템플릿 동결. 매일 발행 부담으로 중단 시 알고리즘 손실 → 5장 단위 사전 제작 + 예약 발행 도구 사용. 인스타 1계정 = 프로필 링크 1개 제약 사전 처리(linktr.ee 또는 도메인 직링크).

### 전략 3 — 네이버 카페 진성 답변 5건 (D-3 ~ D+14)

- **행동**: "쉬운베이킹 클래스" 등 회원수 큰 베이킹 카페 3곳에 D-7부터 가입 → 등업 조건 충족(출석·댓글) → 초보 질문 게시판의 "박력분 뭐예요?", "베이킹파우더 대체 뭘로?" 류 최근 1주일 질문 5건에 진성 답변. 답변 구조 — 질문에 직접 답 4~5줄(앱 데이터 그대로 활용 가능) + 마지막 1줄 "더 자세한 비교는 제가 만든 사전 [링크] 참고하세요". UTM(`?utm_source=naver_cafe&utm_medium=answer&utm_campaign=launch`) 부착.
- **타임라인**: D-7 가입 → D-3 등업 완료 → D+0 ~ D+14 답변 5건 분산 (도배 방지).
- **예상 도달**: 답변 1건당 노출 100~500회, 클릭 5~20건. 5건 누적 25~100명. 첫 10명 달성 가능 시점 D+3~D+7.
- **비용**: 0원 (시간 — 답변 1건 작성 약 15분, 카페 룰 학습 1회 30분).
- **위험·금지선**: 영업·홍보·도배 금지. 답변 본문이 링크보다 짧으면 즉시 스팸 신고 → 강퇴 → 시드 채널 1개 영구 손실. 카페별 등업 조건 사전 확인 필수(미충족 시 답변 권한 없음). 같은 카페에 하루 2건 이상 답변 금지. 자기 카페 만들어 끌고 가려는 시도 금지(MISSION의 Anti-Scope: 커뮤니티 기능 없음).

### 보조 전략 (여력 있으면, 우선순위 외)

- **만개의레시피 댓글 답변**: 인기 베이킹 레시피 댓글에 "박력분 어디서 사요?" 류 질문이 종종 달림. 1건 5분, 누적 효과 작지만 SEO·검색 도달엔 의미.
- **베이킹 클래스·공방 1곳 협업**: 수강 자료 끝에 QR 1개. 1회 노출 20~40명. 협업 성사가 어려우므로 D+14 이후 검토.

## 4. 측정 지표 (첫 10명 도달 검증)

세 가지 층위로 측정. 모두 무료 도구로 가능.

| 층위 | 지표 | 목표 | 측정 방법 |
|---|------|------|---|
| 결과 | 첫 10명 도달 시점 | D+7 이내 | Vercel Analytics or Plausible — unique visitor 누적 10 도달 일자 |
| 결과 | 페르소나 시나리오 도달 | 검색 → 답까지 평균 3초 이내 | 직접 측정 (스톱워치) + 자체 이벤트 로그 |
| 채널 | 채널별 유입 비율 | 3개 전략 모두 0이 아닌 유입 | UTM 파라미터 (utm_source: youtube / instagram / naver_cafe) |
| 채널 | 인스타 카드 1장 → 클릭 | CTR 1% 이상 | 인스타 인사이트 + UTM 클릭 수 |
| 채널 | 카페 답변 1건 → 클릭 | 답변당 평균 5클릭 이상 | UTM 클릭 수 |
| 행동 | 비교 모드 사용률 | 전체 세션 중 10% 이상이 `/compare` 진입 | Vercel Analytics 페이지뷰 |
| 행동 | 재방문율 | 10명 중 2명 이상 7일 내 재방문 | Plausible 재방문 지표 |

7일 시점에 위 표를 채워 — 어느 채널이 살아 있고 어느 채널을 D+14에 두 배로 밀지 결정.

## 5. Open Questions

데모데이 전·직후에 결정해야 할 항목.

- 아내 채널 언급의 톤·노출 강도 — 도입부 한 마디 vs 마무리 한 마디 vs 별도 PR 영상 1편. 아내와 D-3까지 합의.
- 인스타 계정 분리 여부 — 본인 개인 계정 vs 사이트 전용 계정 신설. 신설 시 팔로워 0부터 시작 → 첫 한 달 도달 ↓. 본인 계정 활용 시 톤 일관성 ↓. 결정 필요.
- 네이버 카페 등업 조건 — 카페별 출석·댓글 점수 요건 다름. D-7까지 3곳 모두 사전 조사 + 등업 시뮬레이션.
- linktr.ee vs 직접 도메인 — 인스타 프로필 링크 1개 자리에 어느 쪽을 둘지. 직접 도메인이 분석·신뢰도 모두 우위지만 다중 CTA(추후 v1.5)는 linktr.ee가 유리.
- v1.5에서 외부 베이킹 경험자 리뷰를 얹을 때, 그 리뷰어를 어디서 모집할지 — 아내 네트워크가 1순위, 카페 활동가가 2순위.
