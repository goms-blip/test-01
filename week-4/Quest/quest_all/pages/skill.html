JSON 재료 → 레시피 Skill

재료 1개 = JSON 파일 1개. 폴더에서 읽어 자취생 1인분 15분 레시피를 만들어주는 Claude Code 슬래시 명령. 아래는 현재 Quest 폴더에 등록된 재료들입니다.

현재 등록된 재료 (ingredients/*.json)

사용 흐름

1

재료 JSON 파일 작성

ingredients/egg.json 처럼 재료 1개당 파일 1개. { "name", "quantity", "category" } 구조.

2

슬래시 명령 실행

Claude Code에서 /recipe 입력. 폴더 전체를 스캔합니다.

3

레시피 마크다운 출력

1인분, 15분 이내, 자취생 난이도 레시피가 마크다운 파일로 저장됩니다.

JSON 예시

{
  "name": "계란",
  "quantity": "6개",
  "category": "냉장"
}
