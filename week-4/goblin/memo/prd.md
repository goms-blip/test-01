## [PRD] 포스트잇 스타일 상태 변화 메모장
버전: 1.0
상태: 초안 (Draft)
## 1. 제품 개요 (Product Overview)
본 서비스는 시각적으로 직관적인 '포스트잇' 형태의 UI를 가진 메모 앱입니다. 사용자는 메모의 진행 상태에 따라 색상이 자동으로 변하는 시스템을 통해 할 일의 진척도를 한눈에 파악할 수 있습니다.
## 2. 핵심 가치 및 목표

* 시각적 직관성: 색상 변화(노랑 → 연두 → 회색)를 통해 상태 구분.
* 공간 효율성: 포스트잇이 쌓이는(Stacking/Grid) 구조로 정보 집약도 향상.
* 단순함: 복잡한 설정 없이 상태 전환만으로 태스크 관리.

## 3. 주요 기능 및 사용자 흐름 (Features)

* 메모 생성: 신규 메모 작성 시 기본 노란색(Todo)으로 생성.
* 상태 변경:
* 진행 중(In-Progress) 전환 시 연두색으로 변경.
   * 완료(Done) 전환 시 회색으로 변경.
* 메모 보드: 포스트잇 이미지들이 그리드 또는 리스트 형태로 쌓이는 뷰 제공.

## 4. 데이터베이스 설계 (DB Schema)

| 컬럼명 | 타입 | 설명 |
|---|---|---|
| id | UUID / INT | 메모 고유 식별자 (PK) |
| content | TEXT | 메모 내용 |
| status | ENUM | 상태 값 (TODO, IN_PROGRESS, DONE) |
| color_code | VARCHAR(7) | 상태에 따른 HEX 코드 (#FFFF00, #90EE90, #D3D3D3) |
| created_at | TIMESTAMP | 생성 일시 |
| updated_at | TIMESTAMP | 상태 변경 일시 |

## 5. UI/UX 요구사항 (Design)

* 포스트잇 디자인:
* 그림자 효과(Drop Shadow)를 주어 종이가 붙어있는 듯한 입체감 부여.
   * 상단이나 하단에 살짝 말린 듯한 디테일 추가.
* 컬러 가이드:
1. Yellow (#FFFFE0): 대기 중 (신규 생성 시 기본값)
   2. Light Green (#CCFFCC): 현재 진행 중인 작업
   3. Light Grey (#F5F5F5): 완료된 작업 (텍스트에 취소선 적용 권장)
* 배치: 생성 순서에 따라 좌측 상단부터 차례대로 쌓이는 그리드 방식.
